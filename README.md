# Next.js Application - Product Viewer

This is a simple Next.js application for viewing products.  

## Prerequisites

Make sure you have the following installed on your machine:

- [Node.js](https://nodejs.org/) (v16 or higher recommended)
- [npm](https://www.npmjs.com/)

## Setup & Run

1. **Clone the repository**:

   ```bash
   git clone <https://github.com/asadshiekh/e-commerece-frontend>
   cd <your-project-folder>
   ```

2. **Install dependencies**:

   ```bash
   npm install
   ```

3. **Run the development server**:

   ```bash
   npm run dev
   ```

4. **Open the application in your browser**:

   Visit [http://localhost:3000/products](http://localhost:3000/products) to view the products page.
